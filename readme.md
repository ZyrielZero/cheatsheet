# Module 1: Linux & Git Commands Cheat Sheet

A quick reference to common commands.

## Linux Commands

### ls
Lists files and directories.

### pwd
Prints the current working directory.

### cd
Changes directory.

### mkdir
Creates a directory.

### rm
Removes files.

### cp
Copies files.

### mv
Moves or renames files.

### grep
Searches text by pattern.

### chmod
Changes permissions.

## Git Commands

### git init
Initializes a new repo.

### git add
Stages changes.
